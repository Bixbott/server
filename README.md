BIXBOTT

MISSION

Bixbott là nền tảng AI Agent giúp người dùng xây dựng, quản lý và triển khai phần mềm bằng ngôn ngữ tự nhiên.

Mục tiêu cuối cùng là tạo ra một hệ thống AI có thể hoạt động như một đội ngũ phát triển phần mềm hoàn chỉnh.

---

CORE PRINCIPLES

1. Không tạo file rác.
2. Không tạo repo rác.
3. Không tạo tính năng khi chưa hoàn thành tính năng hiện tại.
4. Mọi thay đổi phải phục vụ mục tiêu của Bixbott.
5. README.md là nguồn sự thật duy nhất.

---

PRODUCT GOALS

Goal 1

AI CTO

Nhiệm vụ:

- Phân tích dự án
- Tạo roadmap
- Tạo sprint
- Đánh giá tiến độ

GitHub Manager

Nhiệm vụ:

- Đọc repository
- Đọc issue
- Tạo issue
- Tạo pull request
- Quản lý project

Goal 3

Project Planner

Nhiệm vụ:

- Chia dự án thành task
- Theo dõi tiến độ
- Đề xuất ưu tiên

AI Developer Team

Bao gồm:

- CTO
- Architect

- QA Engineer

Các agent phối hợp với nhau để hoàn thành dự án.

DEVELOPMENT RULES

Khi xây dựng Bixbott:

1. Luôn sửa hệ thống hiện tại trước khi tạo hệ thống mới.
2. Không tạo repo mới nếu có thể dùng repo hiện tại.
3. Không tạo thư mục mới nếu không thật sự cần thiết.
4. Mọi code phải chạy được trước khi thêm tính năng mới.
5. Mọi tính năng phải có mục đích rõ ràng.
